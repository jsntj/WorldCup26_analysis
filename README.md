# WorldCup26_analysis

## Python environment setup

1. Check Python version:
   ```bash
   python3 --version
   ```
2. Create a virtual environment:
   ```bash
   python3 -m venv .venv
   ```
3. Activate the virtual environment:
   - macOS/Linux:
     ```bash
     source .venv/bin/activate
     ```
   - Windows (PowerShell):
     ```powershell
     .venv\Scripts\Activate.ps1
     ```
4. Upgrade pip:
   ```bash
   python -m pip install --upgrade pip
   ```

When finished:
```bash
deactivate
```
